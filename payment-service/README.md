# Service Paiement

API REST pour le traitement des paiements

## Endpoints

- `POST /api/payments` - Créer un paiement
- `GET /api/payments` - Lister tous les paiements
- `GET /api/payments/:orderId` - Récupérer un paiement

## Installation

```bash
npm install
npm start
```

Le service écoute sur le port 3002
